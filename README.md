# API Rest Boilerplate
A simple RESTful API boilerplate with auto generated admin panel based on Sequelize and Mongoose models using Admin Bro

#### Requirements:
- **NodeJS** > v12.19.0 (recommended: [NVM Github repository](https://github.com/nvm-sh/nvm))
- **Docker** in order to run services

## Run Docker services
Run command:
```docker-compose up```

A MySQL Database service will be available on port 3306

## Start server
Install dependencies: ```yarn install```

Run server on development mode: ````yarn dev````

## Admin panel
Once you are running the server an admin panel will be available at http://localhost:3000/admin
